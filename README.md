# 📝Chrome Extension 🚀

## 🛠️ Cách cài đặt

1. **Clone repo này**:

   ```bash
   git clone https://github.com/lcisme/Extension-Chrome.git
   ```

2. **Cài đặt Backend (NestJS):**:

   ```cd backend
   npm install
   npm run start
   Backend sẽ chạy trên http://localhost:3001.
   ```
